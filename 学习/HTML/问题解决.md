## 1. 英文及数字字符串溢出

* 产生原因：

在一个设定好宽度的div中，当我们输入的中文文字长度超过了设定宽度时，会自动换到下一行。

但是，如果输入纯英文字符或数字时，此时无论你div设定宽度为多少，都是不换行且在同一行输出，实现溢出效果

* 解决方法：

在div的style中设定 word-break:break-all; 允许单词内换行即可实现自动换行

```
div{
	word-break:break-all;
}
```
